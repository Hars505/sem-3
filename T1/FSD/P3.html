HTML

HTML (HyperText Markup Language) is the backbone of every webpage you’ve ever visited.
It’s a markup language, not a programming language,
which means it’s used to structure content on the web—not to make it interactive

CSS

CSS (Cascading Style Sheets) is what gives life and personality to your HTML structure—it’s
the design language of the web. If HTML is the skeleton, CSS is the skin, clothes, and style.

JAVASCRIPT

JavaScript is the engine behind interactive web experiences. If HTML is the structure and
CSS is the style, JavaScript is the brain—it makes websites dynamic, responsive, and intelligent
